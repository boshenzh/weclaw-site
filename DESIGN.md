# OpenClaw 网站设计说明

## 设计概览

这是一个专业的 OpenClaw 中文落地页，采用现代化设计风格，突出 AI 自动化能力和企业级应用场景。

## 视觉设计

### 配色方案
- **主色调**: 蓝色到靛蓝渐变 (#2563EB → #4F46E5)
- **辅助色**: 白色、浅灰色背景
- **强调色**: 紫色、粉色渐变装饰

### Logo 设计
- **OpenClaw Logo**: 圆角方形，蓝色渐变背景，白色 "OC" 字母
- **应用场景**: 导航栏、页脚、工作流示例卡片

## 页面结构

### 1. 导航栏
- 固定顶部，半透明背景，毛玻璃效果
- 左侧: OpenClaw logo + 品牌名
- 右侧: 导航链接（功能、集成、案例、上手流程、FAQ）+ CTA 按钮

### 2. Hero 区域
- 左侧: 标题、描述、双 CTA 按钮、特性标签
- 右侧: 工作流示例卡片，展示合同审阅自动化流程
- 背景: 渐变装饰球体

### 3. 集成平台展示
- 白色背景区域
- 6 个集成平台卡片（飞书、Telegram、Notion、钉钉、企业微信、Slack）
- 悬停效果: 边框变蓝、阴影增强

### 4. 核心功能
- 3 列网格布局
- 每个功能卡片包含: 数字图标、标题、描述
- 悬停效果: 边框变蓝、阴影增强

### 5. 客户案例（律所）
- 蓝色渐变背景
- 左侧: 公司信息、挑战、解决方案
- 右侧: 3 个成果指标，带勾选图标
- 白色卡片容器，圆角设计

### 6. 上手流程
- 3 步流程，网格布局
- 简洁的步骤说明

### 7. FAQ
- 白色背景
- 问答卡片，悬停效果

### 8. CTA 区域
- 蓝色渐变背景
- 居中布局
- 双按钮: 查看文档 + 一键部署

### 9. 页脚
- 白色背景，顶部边框
- 左侧: OpenClaw logo + 品牌名
- 右侧: 导航链接
- 底部: 版权信息

## 交互设计

### 悬停效果
- 卡片: 边框颜色变化、阴影增强
- 按钮: 阴影增强、背景色变化
- 链接: 颜色变化

### 响应式设计
- 移动端: 单列布局
- 平板: 2 列布局
- 桌面: 3-6 列布局

## 品牌元素

### OpenClaw Logo
- 形状: 圆角方形
- 背景: 蓝色到靛蓝渐变
- 文字: 白色 "OC"
- 尺寸: 导航栏 40x40px，页脚 32x32px

### 集成 Logo
- 当前使用占位 SVG
- 建议替换为真实品牌 logo
- 尺寸: 64x64px
- 格式: SVG

## 文案风格

- **标题**: 简洁有力，突出价值
- **描述**: 清晰易懂，面向中文用户
- **CTA**: 行动导向，明确指引
- **案例**: 真实数据，可信度高

## 技术实现

- **框架**: Next.js 16 + React 19
- **样式**: Tailwind CSS
- **字体**: 系统默认字体
- **图标**: SVG 内联
- **动画**: CSS transition
