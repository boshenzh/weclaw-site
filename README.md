基于 setupclaw.com 风格的中文版网站，提供 OpenClaw 的专业部署和托管服务。

## 文案风格

### 对话式、专业但不装腔作势

**示例 1 - 诚实透明**
> "让 AI 访问你的邮件和日历永远不是 100% 安全的。我们做的是将风险降到最低，并提供完全透明的安全措施。"

**示例 2 - 直接了当**
> "你当然可以自己部署。OpenClaw 是开源的。但在安全加固、持续维护、故障排查上，你会花费大量时间。我们的客户通常是创始人和高管，他们的时间更值钱。"

**示例 3 - 具体明确**
> "每 30 分钟自动检查系统，无需手动触发。睡觉时它也在工作。"

## 自定义内容

### 修改价格
编辑 `src/app/page.tsx` 中的 `pricing` 数组：
```typescript
const pricing = [
  {
    name: "托管部署",
    price: "¥20,000",
    description: "推荐方案",
    features: [...],
    recommended: true,
  },
];
```

### 修改飞书会议链接
当前使用的真实飞书预约链接：
```
https://h91srrlmnb.feishu.cn/scheduler/4280da450911da25
```

如需更换，搜索并替换所有该链接为你的新预约链接。

### 修改 FAQ
编辑 `faqs` 数组，添加或修改问题和答案。

### 修改对比表格
编辑 `comparison` 数组，调整对比维度。
