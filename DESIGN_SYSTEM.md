# WeClaw 设计系统

## 品牌标识

### 公司名称
**WeClaw** - 专业法律科技解决方案提供商

### 品牌定位
- 专业、可信、创新
- 面向企业级客户
- 法律科技领域的领导者

## 视觉设计

### 配色方案

#### 主色调
- **黑色** `#18181b` (zinc-900) - 主要文字、导航背景
- **白色** `#ffffff` - 背景、按钮
- **深灰** `#71717a` (zinc-500) - 次要文字
- **浅灰** `#fafafa` (zinc-50) - 区域背景

#### 使用原则
- 以黑白灰为主，体现专业和严谨
- 避免使用鲜艳的颜色
- 保持高对比度，确保可读性

### 字体系统

#### 字体族
```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;
```

#### 字重
- **Light (300)** - 大标题
- **Regular (400)** - 正文
- **Medium (500)** - 小标题
- **Semibold (600)** - 强调
- **Bold (700)** - 导航、按钮

#### 字号层级
- **Hero 标题**: 60px (lg:72px)
- **Section 标题**: 48px (lg:60px)
- **卡片标题**: 24px
- **正文**: 16px
- **小字**: 14px
- **标签**: 12px

### 间距系统

#### 容器
- **最大宽度**: 1280px (max-w-7xl)
- **水平内边距**: 24px (px-6) / 32px (lg:px-8)

#### 垂直间距
- **Section 间距**: 80px (py-20) / 112px (lg:py-28)
- **元素间距**: 16px, 24px, 32px, 48px

### 布局原则

#### 网格系统
- **移动端**: 1 列
- **平板**: 2 列
- **桌面**: 2-4 列

#### 对齐
- 左对齐为主
- 标题和正文左对齐
- 统计数据居中对齐

## 组件设计

### 导航栏
```
- 固定顶部 (fixed top-0)
- 高度: 80px (h-20)
- 白色背景，底部边框
- Logo 左侧，导航右侧
- 移动端隐藏导航菜单
```

### 按钮
```
主要按钮:
- 黑色背景，白色文字
- 无圆角 (rounded-none)
- 内边距: px-8 py-3.5
- 悬停: 背景变浅

次要按钮:
- 白色背景，黑色边框
- 悬停: 背景变黑，文字变白
```

### 卡片
```
- 白色背景
- 边框: 1px solid zinc-200
- 无圆角
- 悬停: 背景变黑，文字变白
- 内边距: p-10 / p-12
```

### 统计数据
```
- 大号数字 (text-4xl / text-5xl)
- 字重: 300 (font-light)
- 小号标签 (text-sm)
- 大写字母 (uppercase)
- 字母间距 (tracking-wider)
```

## 页面结构

### Hero Section
```
- 深色背景 (bg-zinc-900)
- 白色文字
- 大标题 + 描述 + 双按钮
- 垂直内边距: py-24 / lg:py-32
```

### Stats Section
```
- 白色背景
- 4 列网格 (移动端 2 列)
- 居中对齐
- 底部边框分隔
```

### Services Section
```
- 浅灰背景 (bg-zinc-50)
- 2x2 网格布局
- 网格间隙: gap-px
- 卡片悬停效果
```

### Case Study Section
```
- 浅灰背景
- 左侧: 深色侧边栏 (公司信息)
- 右侧: 白色内容区 (挑战、方案、成果)
- 底部: 客户引言
```

### Footer
```
- 白色背景，顶部边框
- 4 列网格 (移动端 1 列)
- 底部: 版权和法律链接
```

## 交互设计

### 悬停效果
```css
transition-colors duration-300
```

#### 链接
- 文字颜色: zinc-600 → zinc-900

#### 按钮
- 背景颜色变化
- 无阴影效果

#### 卡片
- 背景: white → zinc-900
- 文字: zinc-900 → white

### 动画
- 避免复杂动画
- 使用简单的颜色过渡
- 过渡时间: 200-300ms

## 响应式设计

### 断点
- **sm**: 640px
- **md**: 768px
- **lg**: 1024px
- **xl**: 1280px

### 移动端优化
- 单列布局
- 隐藏导航菜单
- 调整字号和间距
- 简化复杂布局

## 内容策略

### 文案风格
- **专业**: 使用行业术语
- **简洁**: 避免冗长描述
- **数据驱动**: 使用具体数字
- **客户导向**: 强调价值和成果

### 标题层级
1. **H1**: 页面主标题 (每页一个)
2. **H2**: Section 标题
3. **H3**: 卡片标题
4. **标签**: 小号大写文字

## 可访问性

### 对比度
- 文字与背景对比度 ≥ 4.5:1
- 大号文字对比度 ≥ 3:1

### 语义化
- 使用正确的 HTML 标签
- 添加 alt 属性
- 使用 aria-label

### 键盘导航
- 所有交互元素可通过键盘访问
- 清晰的焦点状态

## 性能优化

### 图片
- 使用 Next.js Image 组件
- 懒加载
- 响应式图片

### 字体
- 系统字体优先
- 避免加载外部字体

### CSS
- Tailwind CSS 按需生成
- 移除未使用的样式

## 品牌应用

### Logo 使用
- 当前使用文字 logo "WeClaw"
- 可添加图形 logo
- 保持简洁和专业

### 色彩应用
- 主要使用黑白灰
- 避免彩色装饰
- 保持一致性

### 图片风格
- 专业商务照片
- 高质量、高分辨率
- 避免过度修饰
